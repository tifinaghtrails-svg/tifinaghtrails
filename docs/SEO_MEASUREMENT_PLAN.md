# SEO Measurement Plan

## First 7 Days After Deployment

- Confirm production deploy succeeds.
- Confirm `/sitemap.xml` and `/robots.txt` load.
- Submit sitemap in Google Search Console.
- Inspect the home page, tours page, and one tour detail page.
- Check Vercel Analytics for page views and broken routes.

## First 30 Days

- Track impressions and clicks in Search Console.
- Identify pages with impressions but low click-through rate.
- Improve page titles and meta descriptions where CTR is weak.
- Check Speed Insights for mobile LCP and INP.
- Review booking and contact conversion events.

## Monthly

- Add or improve one useful content section based on Search Console queries.
- Refresh tour descriptions with real customer questions.
- Add internal links from FAQs to tours.
- Check for 404s and redirect only when a real replacement exists.
- Review content claims against proof documents.

## Core KPIs

- Indexed pages
- Organic impressions
- Organic clicks
- Average position for target queries
- Booking request submissions
- Contact form submissions
- WhatsApp clicks
- Mobile LCP and INP
