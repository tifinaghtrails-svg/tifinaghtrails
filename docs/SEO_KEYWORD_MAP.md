## Internal Linking Rules

- Link tour cards to the exact tour detail URL.
- Link destination sections to related tours where possible.
- Link FAQ answers back to the most relevant tour pages.
- Use descriptive anchor text such as "Mount Toubkal 2 day trek" instead of "click here".
